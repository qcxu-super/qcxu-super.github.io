
# 0. CF

协同过滤，基于近邻的推荐算法，主要有以下两类：

- 基于用户的协同过滤(User-CF)
- 基于物品的协同过滤(Item-CF)

举个例子：

不知道哪个电影是自己喜欢的或评分高的，这时通常会问“跟自己品味差不多”的朋友，看他有什么电影推荐。

# 1. User-CF

## 1.1 算法思路

给用户推荐：“和他兴趣相投的其他用户”喜欢的物品。所以，先找到“相似用户”，再找到“他们喜欢的物品”。

那怎么衡量用户的相似性呢？通过`不同用户`对`相同物品`的评分或偏好程度！
